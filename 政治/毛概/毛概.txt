马克思主义中国化命题的提出、内涵及其理论成果
提出 : 1938年毛泽东在党的六届六中全会上<论新阶段>提出
内涵:
将马克思主义内涵同中国的实际情况相结合, 用马克思主义思想解决中国的实际问题
马克思主义中国化就是总结中国革命, 建设的成功经验,为马克思主义添砖加瓦
将马克思主义用中国百姓喜闻乐见的方式阐述, 使之成为有中国特色的马克思主义
成果 : 毛泽东思想, 中国特色社会主义思想


毛泽东思想的产生条件、发展过程和历史地位
产生条件 :
时间背景 : 十月革命, 中国局势动荡
思想背景 : 马克思列宁主义
阶级基础 : 工农阶级壮大
发展过程 :
[萌芽] : <湖南农民运动考察报告> <中国社会各阶级的分析>
[发展] : <星星之火, 可以燎原><中国红色政权为什么能存在><反对本本主义><井冈山斗争> : 提出农村包围城市, 武装夺取政权
[成熟] : <实践论>和<矛盾论> ...: 1945中共七大将毛泽东思想写入党章
历史地位 :
马克思主义中国化的第一个重要理论成果
是中国革命和建设的科学指南
中国共产党和中国人民的宝贵财富
毛泽东思想活的灵魂 :
1. 独立自主 2. 群众路线 3. 实事求是


邓小平理论的产生条件、基本问题、发展过程、主要内容和历史地位
产生条件 :
时代背景 : 和平和发展成为时代的主题
历史依据: 社会主义建设的经验教训
现实依据: 改革开放和现代化建设
基本问题 : 什么是社会主义, 建设怎么样的社会主义
发展过程 :
[1978  十一届三中全会] :
重新确立解放思想实事求是的思想路线, 停止使用"以阶级斗争为纲", 确定全党的工作中心为社会主义现代化建设
[1982  党的十二大] : 
正式提出建设中国特色社会主义(把马克思主义的普遍真理同中国实际情况相结合, 建设由中国特色的社会主义)
[1987  党的十三大] :
第一次比较系统的阐述了我国社会主义初级阶段理论, 明确概括和全面阐述了党的 "一个中心, 两个基本点" 的基本路线
[1992  南方谈话]
推动改革开放和社会主义现代化建设进入新阶段
社会主义本质是解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕
三个有利于标准
社会主义可以搞市场经济
革命是解放生产力，改革也是解放生产力
[1992  党的十四大]
总结了改革开放14年取得的巨大成就, 初步回答了如何建设社会主义, 巩固和发展社会主义
[1997  党的十五大]
邓小平理论写入党章 (1999 写入宪法)

邓小平理论活的灵魂 : 解放思想, 实事求是的思想路线


邓小平理论补充 
一个中心，指以经济建设为中心
两个基本点 : 指坚持四项基本原则，坚持改革开放

四项基本原则：
坚持社会主义道路
坚持人民民主专政
坚持中国共产党的领导
坚持马列主义、毛泽东思想

三个有利于：
有利于发展社会主义社会的生产力
有利于增强社会主义国家的综合国力
有利于提高人民的生活水平

改革中始终坚持两条根本原则：一是以社会主义公有制经济为主体，一是共同富裕
